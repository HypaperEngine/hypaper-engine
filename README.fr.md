# Hypaper Engine

> Moteur de wallpapers animés pour Hyprland — propulsé par Rust, natif Wayland, piloté par la communauté.

![CI](https://github.com/HypaperEngine/hypaper-engine/actions/workflows/ci.yml/badge.svg)
![Licence](https://img.shields.io/badge/licence-MIT%20OR%20Apache--2.0-blue)

---

## Fonctionnalités

- 🎨 Format de scène custom `.hyscene` — calques, shaders, particules
- ⚡ Rendu GPU via `wgpu` et shaders WGSL
- 🎵 Audio ambient avec visualisation réactive (V2)
- 🖥️ Intégration Hyprland native — workspaces, parallaxe, multi-moniteur
- 🦀 100% Rust — sûr, rapide, fiable
