# Hypaper Engine

> Animated wallpaper engine for Hyprland — Rust-powered, Wayland-native, community-driven.

![CI](https://github.com/HypaperEngine/hypaper-engine/actions/workflows/ci.yml/badge.svg)
![License](https://img.shields.io/badge/license-MIT%20OR%20Apache--2.0-blue)

---

## Features

- 🎨 Custom scene format `.hyscene` — layers, shaders, particles
- ⚡ GPU rendering via `wgpu` and WGSL shaders
- 🎵 Ambient audio with reactive visualization (V2)
- 🖥️ Native Hyprland integration — workspace-aware, parallax, per-monitor
- 🦀 100% Rust — memory safe, fast, reliable
